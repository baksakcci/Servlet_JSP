# Servlet_JSP
Servlet과 JSP 실습
Servlet과 JSP는 JavaEE의 웹 표준으로써 현재까지도 사용되고 있는 WAS의 핵심 기술이다.

이번 실습을 통해서 최소한의 Servlet Application만을 만들어 Spring MVC에 사용되는 Servlet을 심층적으로 알기 위한 내부 동작 방식을 빠르게 이해하는 것이 목적이다.
